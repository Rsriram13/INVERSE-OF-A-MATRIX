### Step 2: Get input form the user

### Step 4: End program

## Program:
#Program to find the inverse of a matrix.
#Developed by: Sriram R
#RegisterNumber:23004952
```
import numpy as np
A=np.array([[6,2,3],[3,1,1],[10,3,4]])
inverse=np.linalg.inv(A)
print(inverse)
```
